# Data Visualization

## Purpose
Helps build effective data visualizations — picking the right chart, designing for readability, and turning raw data into clear stories. Covers static charts, interactive dashboards, and exploratory data analysis.

## When to use
- Choosing the right chart type for a dataset
- Building a dashboard for business or product metrics
- Polishing a chart for a presentation or report
- Exploring an unfamiliar dataset
- Replacing a confusing chart with something clearer

## Approach
- Pick the chart type from the question, not the data shape
- Avoid 3D, dual axes, and pie charts with more than 4 slices
- Label axes clearly and start counts at zero
- Use color with intent — categorical, sequential, or diverging
- Annotate the insight directly on the chart, not in a caption
