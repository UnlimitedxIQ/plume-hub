# Presentations

## Purpose
Turns ideas, briefs, or outlines into structured slide decks that work both as visual aids and as standalone artifacts. Optimized for student talks, classroom presentations, and short professional pitches.

## When to use
- The user has an outline and wants slide content
- A talk needs to be cut down to fit a time limit
- The user wants speaker notes generated alongside slides
- A deck needs to be reorganized for stronger narrative flow
- The user wants a one-slide summary of a longer document

## Approach
- Anchor the deck on a single core message
- One idea per slide — never crowd
- Use the title bar to carry the main takeaway, not just a label
- Pair every claim slide with an evidence or example slide
- Generate concise speaker notes for delivery
