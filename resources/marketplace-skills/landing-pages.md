# Landing Pages

## Purpose
Builds high-converting landing pages from scratch or from a brief — hero, features, social proof, FAQ, and CTAs. Combines proven copy frameworks (PAS, AIDA) with clean, modern visual design.

## When to use
- Building a landing page for a new product or feature
- Improving a page that gets traffic but doesn't convert
- Creating a waitlist or early-access page before launch
- Building a page for a paid ad campaign
- Refreshing an outdated marketing page

## Approach
- Start with a clear value proposition above the fold
- Add social proof (logos, testimonials, stats) within the first scroll
- Use feature blocks with benefit-led headlines, not feature names
- Include an FAQ section to handle objections
- Close with a CTA that matches the visitor's likely intent
