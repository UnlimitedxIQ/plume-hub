# Financial Analysis

## Purpose
Performs structured financial analysis on companies — ratio analysis, valuation, budget variance, and forecast construction. Designed for finance students, analysts, and founders evaluating their own metrics.

## When to use
- Evaluating a company's financials from a 10-K or earnings release
- Building a DCF or comparables-based valuation
- Calculating profitability, liquidity, and leverage ratios
- Constructing a budget variance report
- Building a rolling forecast from historical data

## Approach
- Pull or load the underlying financial statements first
- Calculate the standard ratio set (margins, ROIC, leverage, liquidity)
- Compare against historical trend and peer benchmarks
- Build the valuation model with explicit assumptions
- Surface the 2-3 numbers that actually matter for the decision
