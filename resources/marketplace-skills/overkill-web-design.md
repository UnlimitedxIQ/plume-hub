# Overkill Web Design

## Purpose
Turns a normal website into a ridiculously polished, creative, award-worthy experience. Combines 3D, motion, micro-interactions, and obsessive UX detail. Use when a project needs to look like a $50K agency build instead of a template.

## When to use
- Polishing a portfolio or launch site to award-show level
- Adding cinematic motion and detail to a generic page
- Building a hero section that immediately impresses
- Refining micro-interactions and transitions
- Going from "fine" to "wow" on every detail

## Approach
- Audit every interaction — every hover, click, and scroll should feel intentional
- Add cinematic transitions between sections, not just fades
- Layer motion: subtle background, medium foreground, bold focus
- Use real photography and original illustration over stock
- Sweat the details: cursor states, loading states, empty states, error states
