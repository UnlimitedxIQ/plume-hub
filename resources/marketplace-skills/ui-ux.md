# UI/UX

## Purpose
Guides interface and interaction design — wireframing, user flows, visual hierarchy, accessibility, and usability heuristics. Use whenever shaping a new feature, redesigning an existing screen, or critiquing an interface.

## When to use
- Designing a new screen, modal, or flow
- Critiquing an existing interface for usability issues
- Choosing between layout options for the same content
- Improving accessibility and contrast
- Building a wireframe before handing off to engineering

## Approach
- Start from the user's goal, not the interface elements
- Map the user flow before drawing pixels
- Apply Nielsen's usability heuristics as a checklist
- Establish a clear visual hierarchy with type scale and spacing
- Test with real users early — even paper sketches catch most issues
