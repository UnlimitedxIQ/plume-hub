# Resume Guide

## Purpose
Helps users construct or revise resumes that pass ATS filters and impress human reviewers. Covers bullet phrasing, quantification, section ordering, and tailoring to a specific role. Especially useful for students, career changers, and anyone updating after a long gap.

## When to use
- The user wants a fresh resume from scratch
- An existing resume needs to be tailored to a specific job posting
- Bullets feel weak and need stronger verbs and metrics
- The user needs help reordering sections (experience vs education vs projects)
- An ATS keeps rejecting an application and the user wants to fix the format

## Approach
- Start with the target job description and reverse-engineer keywords
- Lead each bullet with an action verb plus a measurable result
- Keep formatting ATS-safe (no tables, no columns, standard headings)
- Trim everything older than ~10 years unless truly relevant
- End with a one-line summary if the user has a strong narrative
