# Study Flashcards

## Purpose
Turns dense study material into a clean set of question/answer flashcards optimized for active recall. Designed for students cramming for exams, learning vocabulary, or building long-term retention through spaced repetition.

## When to use
- The user pastes a chapter or lecture notes and asks for flashcards
- An exam is coming up and the user wants quick recall practice
- The user is learning a new vocabulary set (foreign language, jargon, definitions)
- The user wants to convert a study guide into Anki / Quizlet format
- A concept-heavy lecture needs to be broken into atomic Q/A pairs

## Approach
- Extract atomic facts, definitions, and cause-effect relationships
- Phrase questions to test understanding, not just recognition
- Keep answers short — one fact per card
- Group cards by topic for easier batching during review
- Output as plain text or CSV for easy import into Anki / Quizlet
